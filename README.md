# BlackJack Band Website Clone

This is a clone of the [BlackJack Band website](https://blackjackband.se) built with Next.js and Tailwind CSS v4.

## Features

- Single-page application with smooth scrolling
- Responsive design
- Built with Next.js and Tailwind CSS v4
- Sections for news, music, tour dates, contact information, and press photos

## Getting Started

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/` - Next.js app router files
- `src/components/` - React components for each section of the site
- `public/images/` - Images used in the website

## Adding Your Own Content

Before running the site for production, you'll need to:

1. Replace the placeholder images in `public/images/` with your own images
2. Update the content in each component with your own text
3. Customize the styling to match your branding

## Deployment

You can deploy this site to any hosting platform that supports Next.js applications, such as Vercel, Netlify, or AWS Amplify.
